# VibeCoding – Landing Page Fitness

## Temática
Landing page sobre gimnasio, nutrición y estilo de vida fitness.

## PROMPT 1
Quiero que me generes una landing page completa usando solo HTML, CSS y JavaScript (vanilla, sin frameworks).

La landing debe ser totalmente responsive y adaptarse correctamente a escritorio, tablet y smartphone.

La temática es: “Gimnasio, nutrición y estilo de vida fitness”.

Quiero que la landing incluya:

1. Una sección hero con:
   - Un título motivador relacionado con fitness.
   - Un subtítulo breve.
   - Una imagen o icono relacionado con entrenamiento o nutrición.

2. Una sección con gráficas que muestren:
   - Comparativa entre calorías quemadas por tipo de entrenamiento (fuerza, cardio, HIIT, movilidad).
   - Evolución semanal de progreso (por ejemplo: peso levantado, minutos de cardio, calorías quemadas).
   - Todo generado con JavaScript vanilla, sin librerías externas.

3. Una sección de nutrición con:
   - Tarjetas de alimentos saludables.
   - Información básica: calorías, proteínas, carbohidratos y grasas.
   - Un diseño visual atractivo y moderno.

4. Una sección interactiva donde el usuario pueda:
   - Introducir su peso, altura, edad y nivel de actividad.
   - Calcular su TMB (Tasa Metabólica Basal) y calorías recomendadas.
   - Mostrar el resultado con una pequeña gráfica o indicador visual.

5. Un menú superior fijo con enlaces a cada sección.

6. Estética:
   - Moderna, limpia y profesional.
   - Colores relacionados con fitness: tonos oscuros, verdes, azules o naranjas.
   - Tipografías claras y legibles.

7. Estructura del proyecto:
   - index.html
   - styles.css
   - script.js

Quiero que me entregues el código completo de los tres archivos, listos para copiar y pegar.
